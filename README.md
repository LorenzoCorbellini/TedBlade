
# TedBlade

1.  **Utenti:**  persone interessate ad analizzare i dati dei contenuti di TED, in particolare speaker, organizzatori e utenti curiosi.
2.  **Valore aggiunto:**  integriamo in un’unica piattaforma dati provenienti dal dataset TED e da YouTube, rendendoli facilmente esplorabili tramite un’interfaccia chiara o accessibili via API.
3.  **Bisogno soddisfatto:**  offrire uno strumento semplice per consultare e analizzare le performance degli speaker e dei loro video, senza dover raccogliere e incrociare manualmente dati da fonti diverse.
